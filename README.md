# Reto1_DIW
Proyecto de una web para un restaurante para el modulo de DIW, 2º de DAW
En este reto se plantean las siguientes actividades:
• Creación de guía de estilo. Colores, iconos, logos, etc.
• Implementación inicial de la interfaz
• Implementación del sitio web en plataformas CMS
• Implementación del sitio web en código (HTML + CSS + jQuery)
• Presupuesto final de la implementación
• Presentación del Proyecto planteado y su solución
